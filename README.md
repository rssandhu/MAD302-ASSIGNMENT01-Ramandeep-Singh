## Project Description
This Android app displays a list of courses using RecyclerView on the main screen. Each course item shows the course code and name. When the user taps a course, the app opens a second screen (DetailActivity) that displays the course code, name, and description passed via Intent extras. The app demonstrates:
- RecyclerView with a custom Adapter and ViewHolder.  
- Item click handling and data transfer using Intent extras.  
- Two‑screen navigation in Android.

## How to run
- Open the project in Android Studio.  
- Build and run on an emulator or device.  
- The main screen shows the course list; tap any course to open the detail screen.
